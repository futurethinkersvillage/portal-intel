# Administration

- We need to be able to assign administrators in the Intel dashboard so that they can have their own administrative view.
- Administrators should have, on any feed page or anywhere they see scraped results:
  - an upvote button
  - a downvote button
  - a comment area where they can give feedback on that individual feed item
- Upvote and downvote buttons should not work unless the comment box contains text.
- There must always be a comment explaining why the feed item was upvoted or downvoted.
- When an item is downvoted, it should be automatically removed from the feed.
- The scraper tool should always be learning from feedback so that it improves its results with every successive scrape.
- In the admin panel we should also track all downvoted items so that we can see a list of what was removed from the feed.
